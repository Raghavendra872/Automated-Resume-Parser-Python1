# Resume Parser

## Description

This project is a Resume Parser developed using Python. It extracts important information from a resume and stores it in an SQLite database.

## Features

- Extracts candidate name
- Extracts email address
- Extracts phone number
- Identifies technical skills
- Saves candidate information into SQLite database

## Technologies Used

- Python
- Regular Expressions (re)
- SQLite

## Project Files

- resume_parser.py
- sample_resume.txt
- candidates.db
- README.md

## Output

The program extracts:
- Name
- Email
- Phone Number
- Skills

and stores them in the SQLite database.
